# Enterprise Performance Tracking Solution

## Overview

Enterprise Performance Tracking Solution is an internal reporting and performance management application developed while working at Western Governors University.

The project was created to solve a real operational need by automating employee metric calculations, simplifying reporting workflows, and improving visibility into individual and team performance.

Unlike a classroom assignment, the application evolved through continuous collaboration with upper management. I regularly demonstrated new functionality, gathered stakeholder feedback, prioritized requested changes, and refined the application through multiple development iterations before it was adopted across multiple departments.

---

## Key Capabilities

- Automated employee metric calculations
- Weekly and daily performance tracking
- Goal progress monitoring
- Cross-sheet data aggregation
- Reporting dashboards
- Workflow automation
- Reduced manual calculations and repetitive reporting

---

## Technologies

- Microsoft Excel
- Advanced Excel formulas
- Data validation
- Conditional formatting
- Cross-worksheet references

## Development Process

This project was developed through an iterative process similar to a professional software development lifecycle.

Throughout development I worked directly with upper management to:

- Gather requirements
- Demonstrate working prototypes
- Collect stakeholder feedback
- Prioritize requested enhancements
- Refine the application through multiple iterations
- Improve usability based on real-world usage

Working directly with stakeholders reinforced the importance of communication, iterative development, and designing software around business needs rather than assumptions.
