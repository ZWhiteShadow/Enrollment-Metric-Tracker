# Enterprise Performance Tracking Solution

## Overview

Enterprise Performance Tracking Solution is an internal business application developed to streamline employee performance reporting and goal tracking.

The project originated from a real operational need while working at Western Governors University. Rather than being built from a predefined assignment, the application evolved through continuous collaboration with upper management to improve reporting accuracy, automate metric calculations, and provide better visibility into employee performance.

Throughout development, I regularly demonstrated new functionality, gathered stakeholder feedback, prioritized requested changes, and iteratively refined the application. The final solution was adopted across multiple departments and became part of daily reporting workflows.

---

## Features

- Automated employee metric calculations
- Goal tracking and progress monitoring
- Simplified reporting workflows
- Improved reporting accuracy through automation
- Business-focused interface designed around real user needs
- Reduced manual calculations and repetitive reporting tasks
